# Condições

## Comandos
`if (condicao) {
    true
} else if (condicao 2) {
    true if the previous statement is false.
} else {
    true if everything is false
}

switch (condicao) {
    case valor 1:
        bloco  
        break     
    case valor 2:
        bloco
        break
    case valor 3:
        bloco
        break
    default:
}`

## Tipos de Condição
*Condição simples* = apenas if.

*Condição composta* = if e else.

*Condições aninhadas* = if, else if, else.

*Condição Múltipla* = switch case.
